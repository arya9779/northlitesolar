# NorthLite Solar - Premium Enterprise Platform

High-performance digital infrastructure for **NorthLite Solar Limited**, a leading renewable energy provider delivering scalable, custom solar PV, battery storage, and smart financing models across West Africa (powering factories, agribusinesses, hotels, and public facilities).

This platform represents a comprehensive, modern visual and architectural revamp of the brand's digital presence, engineered for high responsiveness, premium aesthetics, and interactive user engagement.

---

## 🌟 Visual & Interactive Features

### 1. Turnkey Technical Services Hub (`services.html`)
* **Interactive Divisions Panel**: Clients can dynamically switch between the **5 primary operational divisions**:
  1. *Engineering & Procurement* (Feasibility, Smart Sizing, Tier-1 Component Sourcing).
  2. *Construction & Installation* (Certified technicians, rigorous safety, output optimization).
  3. *Project Finance Options* (Impact-investor backed PPAs and Lease-to-own models).
  4. *Comprehensive Operation & Maintenance* (Routine inspections, physical audits, 24/7 telemetry monitoring).
  5. *Consulting & Advisory* (Market entry, Technical Due Diligence, ROI business cases).
* **Methodology Timeline**: A 5-phase visual pathway tracking project lifecycles:
  `Assess (Audit)` ➔ `Design (Sizing)` ➔ `Finance (Contracts)` ➔ `Build (Construction)` ➔ `Operate (O&M)`.

### 2. High-Fidelity Contact & Consultation Portal (`contact.html`)
* **Interactive Contacts Directory**: Seamless, glass-morphic anchors with instant `mailto:` and `tel:` micro-animations.
* **Corporate HQ Cards**: Styled locations grid detailing the Bolgatanga Head Office and Accra Project Office with complete addresses.
* **Request Consultation Form**: A fully styled consultation intake form with a custom **Verification Success Modal** overlay that displays on successful submission.

### 3. Unified Layout Architecture
* **Global Navigation Header**: A consistent, responsive navigation menu across all files with contextual `active-link` indicator styles:
  `Home` ➔ `Who We Are` ➔ `Solutions` ➔ `Services` ➔ `Contact`
* **Mobile Drawer Overlay**: Staggered transition drawer with staggered entry micro-animations (`drawer-link-stagger`) to guarantee excellent performance on mobile viewports.
* **Premium Footer Columns**: Complete deep-linking columns including brand summaries, dynamic solution quick links, and synchronized corporate HQ directories.
* **Visual Decoupling**: 100% unique visual assets mapped to specific sections to avoid repeated images.

## 🛠️ Technical Stack

* **Structure**: Semantic HTML5 markup.
* **Styling**: TailwindCSS integration tailored with a premium harmonized color system (Primary Gold, HSL Slate-deep, Eco-growth emerald, and translucent Glassmorphism variables).
* **Typography**: Montserrat (for headers and display text) paired with Outfit (for body copy and navigation).
* **Logic**: Vanilla JavaScript for modular events, slideshow transitions, telemetry state counters, and coordinates-tracking spotlight mouse-glow coordinates.

## 🚀 Local Development

### Prerequisites
Make sure you have Node.js and npm installed on your system.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/arya9779/northlitesolar.git
   cd northlitesolar
   ```
2. Install local http-server dependency:
   ```bash
   npm install
   ```

### Running Locally
To launch the development server with **browser caching completely disabled** (preventing static page files from being aggressively cached locally during development):
```bash
npm run dev
```

The terminal will launch a local server. Navigate to:
* **Local Server**: [http://localhost:8080](http://localhost:8080)
